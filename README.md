Inception
A Docker-based infrastructure project from the 42 curriculum that sets up a small virtualized network of services.

Description
This project involves creating a professional infrastructure using Docker containers. The setup includes multiple services running in isolated containers that communicate with each other, simulating a real-world web application environment.

Services
NGINX - Web server with TLS/SSL termination

WordPress - Content management system with php-fpm

MariaDB - MySQL database server
